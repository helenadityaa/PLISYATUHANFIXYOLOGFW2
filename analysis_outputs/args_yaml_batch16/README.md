# Batch 16 args.yaml archive

Folder ini berisi salinan `args.yaml` dari run YOLO batch 16 saja.
File dari `batch_size_8` sengaja tidak disalin dan tidak diupload.

Struktur di `files/` mengikuti sumber setelah folder `runs/`, misalnya:

- `files/detect/batch_size_16/.../args.yaml` untuk training
- `files/detect_test/batch_size_16/.../args.yaml` untuk evaluasi/test
- `files/detect_test/split_70_15_15/batch_size_16/.../args.yaml` untuk salinan test split 70/15/15

Lihat `manifest.csv` untuk mapping sumber run ke file export.
