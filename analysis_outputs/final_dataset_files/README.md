# Final dataset evidence files

Folder ini berisi artefak kecil untuk laporan, bukan seluruh dataset gambar.

- `internal_80_10_10_seed7_data.yaml`: data.yaml dataset internal final yang dipakai model terbaik.
- `internal_70_15_15_seed7_data.yaml`: data.yaml split pembanding 70:15:15.
- `internal_60_20_20_seed7_data.yaml`: data.yaml split 60:20:20 yang baru diprepare.
- `external_opensarship1_data.yaml` dan `external_opensarship2_data.yaml`: data.yaml external test jika folder sumber tersedia lokal.
- `yolo_label_examples/`: contoh label YOLO `.txt` untuk kelas Fishing, Cargo, Passenger.
